Installation
---

```bash
npm install eastern eastern-cli --save-dev
# or
yarn add -D eastern eastern-cli
```

The `eastern` command defines `global.spec`, `global.it`, `global.describe`.
And find and run `test.mjs` / `test/**/*.mjs`.

```bash
eastern
# 
# 
#   0 passing (5 ms)
```
